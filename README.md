# 家計簿管理アプリ（Expense Tracker）

このリポジトリは、日々の支出を記録・集計する CLI ベースの家計簿アプリです。  
SQLite を使用してデータを永続化します。

## 機能

- 支出の追加  
- 支出一覧の表示  
- 月別合計表示  
- カテゴリ別合計表示

##  使い方

### 1. 支出の追加
(例)python main.py add 2025-01-10 食費 800 ランチ
### 2.支出一覧表示の使い方
(例)python main.py list
### 3.月別合計表示の使い方
(例)python main.py month 2025-01
### 4.カテゴリ別合計表示の使い方
(例)python main.py category
